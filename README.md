# Web Portfolio

A personal portfolio website built with HTML, CSS, Bootstrap 5, and vanilla JavaScript. 
The site introduces me, showcases my education, projects, and interests, and provides a way for visitors to get in touch.

Live Site: [https://acherpatmos.github.io/My-portfolio-website/#Home](url)


## Overview

The portfolio is a single-page site (index.html) with the following sections:


Home : Sticky navbar with responsive collapse for smaller screens

About : Profile photo, name, and short intro.

Education : Current degree program and areas of study

Projects : Showcase of practice/course projects.

Hobbies & Interests : Auto-playing Bootstrap carousel of personal interests

Contact : Social/contact links (LinkedIn, email, GitHub) plus a message form with client-side validation


## Built With

HTML5 for semantic page structure

CSS3 for custom styling on top of Bootstrap

Bootstrap 5 for responsive grid, navbar, cards, carousel, and utility classes

JavaScript for form validation and interactive alerts

Git & GitHub for version control

## Features

Responsive navigation : collapses into a toggler menu on smaller screens, stays centered on large screens

Bootstrap carousel : showcases hobbies and interests with captions

Contact section : LinkedIn, email, and GitHub links that open in a new tab (target="_blank" with rel="noopener noreferrer" for security)

Form validation : JavaScript checks that name, email, and message fields are filled before confirming a message was "sent," with specific prompts for whichever fields are missing
